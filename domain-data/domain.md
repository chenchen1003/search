# Domain: Nike 产品目录

## What this index contains
本索引包含 Nike（耐克）及 Jordan 品牌的运动产品信息，涵盖跑步鞋、篮球鞋、足球鞋、休闲鞋、运动服装（T恤、卫衣、短裤、裤子）、运动装备及配件（背包、球、护具等）。每条记录包含商品名称、货号、品类描述、商品类型、配色和适用性别等字段。

## Answerable queries
- 特定类型的鞋款（篮球鞋、跑步鞋、足球鞋、板鞋等）
- 特定系列或型号（Air Max、Dunk、Jordan、Pegasus 等）
- 按性别/年龄筛选（男、女、儿童、大童）
- 按运动场景查询（跑步、训练、篮球、足球、高尔夫）
- 特定颜色或配色的商品
- 运动服装（卫衣、外套、短裤）和运动配件（包、球）

## Not answerable
- 电影、音乐、书籍等娱乐内容
- 非 Nike/Jordan 品牌的产品
- 定价、库存、物流等商务信息
- 历史事件、地理、科学等通用知识
- 食品、电子产品等与运动无关的品类
